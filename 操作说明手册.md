## 如何判断应用已经真正启动成功

不要只看页面是否打开，至少要同时确认：

- Tauri 窗口正常显示，不是空白页
- `tauri-dev.log` 中出现：
  - `Global sidecar started on port 31415`
  - `TCP health check passed`
  - `GET http://127.0.0.1:31415/... -> 200`
  - `No update available, already on latest version`

首启无报错的核心检查命令：

```powershell
Select-String -Path .\tauri-dev.log -Pattern '\[ERROR\]|error:'
```

本次结论：

- 未发现应用级 `ERROR`
- `NativeCommandError` 只是 PowerShell 包装子进程输出的形式，不代表应用失败
- `No proxy configured` 只是继承系统网络配置的普通信息，不是错误

### 2. portable 包的正确判定标准

不要只看 ZIP 是否存在，也不要只看里面有没有 `myagents.exe`。

至少还要确认 ZIP 内包含这些关键运行时条目：

- `server-dist.js`
- `plugin-bridge-dist.mjs`
- `nodejs\`
- `cli\`
- `mino\`
- `bundled-skills\`
- `claude-agent-sdk\`
- `sharp-runtime\`
- `tsx-runtime\`
- `vcruntime140.dll`
- `vcruntime140_1.dll`

如果只有 `myagents.exe` 和两个 VC++ DLL，说明这是残缺 portable 包，不应视为可交付产物。
